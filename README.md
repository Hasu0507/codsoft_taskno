Task 1: To-Do List 
A To-Do List is a simple GUI application developed using Python’s Tkinter library for organizing daily tasks efficiently. Rather than using paper notes, this application enables tasks to be inserted, deleted, or viewed digitally. The project consists of referencing various modules, designing functions for the application, designing the primary GUI window, storing tasks either on a database or a list, or adding GUI components with associated events.

Task 2: Calculator
The Calculator application is a simple GUI-based application developed using the “Tkinter” library. The application includes input spaces and buttons for mathematical operations such as addition, subtraction, multiplication, and division. The application also includes functions for the buttons and calculating the result of the operation. The “Tkinter” library provides an easy-to-use GUI for the application.

Task 3: Password Generator
The Password Generator is a graphical user interface application developed using Tkinter in Python programming to create strong and random passwords based on the input given by the user in terms of password length. The password is created using random characters with all the needed functions like labels, entry boxes, and the button to create the password displaying immediately in the output box.
